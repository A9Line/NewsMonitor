# Система мониторинга и публикации отраслевых новостей

Программа собирает новости из открытых источников, отбирает нужные по темам
и публикует их в Telegram. Написана на C# (.NET 10).

## Что умеет

- Собирает новости сразу из нескольких источников через RSS
- Достаёт заголовок, дату, описание, ссылку и теги
- Фильтрует новости по заданным темам (ИТ, автоматизация, энергетика и т.д.)
- Ограничивает количество новостей в день через конфиг
- Публикует готовый дайджест в Telegram

## Источники

- Habr
- CNews
- iXBT
- ЭнергоНьюс

## Как устроено

Логика сбора и логика отправки разделены, парсинг не зашит внутрь бота.

- Сбор новостей живёт в папке `Collectors`. Каждый источник это отдельный класс.
  Все они работают через общий интерфейс `INewsCollector`, поэтому добавить новый
  источник можно одним маленьким классом, ничего больше не меняя.
- Отправка живёт в папке `Publishers` за интерфейсом `INewsPublisher`. Сейчас это
  Telegram, но при желании можно дописать другой способ доставки и подключить его,
  не трогая остальной код.
- `Program.cs` связывает всё вместе: читает конфиг, собирает новости, фильтрует,
  отбирает нужное количество и отдаёт на публикацию.

## Настройка

Все настройки в файле `config.json`:

- `MaxNewsPerDay` — сколько новостей публиковать за раз
- `Topics` — список тем для фильтрации
- `Sources` — какие источники использовать
- `Telegram.Token` и `Telegram.ChatId` — данные бота

## Запуск

1. Открыть проект в Visual Studio
2. Вписать свой токен бота и chat_id в `config.json`
3. Запустить (F5)

## Примечание

Для отправки в Telegram локально нужен VPN, потому что api.telegram.org из России
напрямую не открывается.
